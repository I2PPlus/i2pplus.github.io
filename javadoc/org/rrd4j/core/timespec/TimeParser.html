Package org.rrd4j.core.timespec

Class TimeParser

java.lang.Object
org.rrd4j.core.timespec.TimeParser
public class TimeParser
extends Object
Class which parses at-style time specification (described in detail on the rrdfetch man page), used in all RRDTool commands. This code is in most parts just a java port of Tobi's parsetime.c code. For years written with two digits, any year before 38 will be post 2000.
