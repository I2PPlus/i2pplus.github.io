Package org.rrd4j.core

Interface RrdFileBackend

All Known Implementing Classes:
RrdNioBackend, RrdRandomAccessFileBackend, RrdSafeFileBackend
public interface RrdFileBackend
An abstract backend which is used to store RRD data to ordinary files on the disk.

Every backend storing RRD data as ordinary files should inherit from it, some check are done in the code for instanceof.
