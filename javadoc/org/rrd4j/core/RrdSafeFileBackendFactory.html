All Implemented Interfaces:
Closeable, AutoCloseable
@RrdBackendAnnotation(name="SAFE",
                      shouldValidateHeader=true,
                      cachingAllowed=false)
public class RrdSafeFileBackendFactory
extends RrdRandomAccessFileBackendFactory
Factory class which creates actual RrdSafeFileBackend objects.

Because of locking, each RrdDb can be open only once even from within the JVM. So usage of the RrdDbPool is mandatory with this backend.
