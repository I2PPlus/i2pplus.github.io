Class SubscriptionIterator

java.lang.Object
net.i2p.addressbook.SubscriptionIterator
All Implemented Interfaces:
Iterator<AddressBook>
class SubscriptionIterator
extends Object
implements Iterator<AddressBook>
An iterator over the subscriptions in a SubscriptionList. Note that this iterator returns AddressBook objects, and not Subscription objects. Yes, the EepGet fetch() is done in here in next().
