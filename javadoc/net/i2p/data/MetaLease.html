Class MetaLease

public class MetaLease
extends Lease
Like a Lease, but points to another LeaseSet. TunnelId unused. Flags unsupported. The LeaseSet Hash is available at getGateway() / setGateway(). Length is 40. PRELIMINARY - Subject to change - see proposal 123
