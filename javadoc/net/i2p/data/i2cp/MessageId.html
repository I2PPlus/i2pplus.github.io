Class MessageId

java.lang.Object
net.i2p.data.i2cp.MessageId
public class MessageId
extends Object
Defines the message ID of a message delivered between a router and a client in a particular session. These IDs are not globally unique. As of 0.9.48, does NOT extend DataStructureImpl.
